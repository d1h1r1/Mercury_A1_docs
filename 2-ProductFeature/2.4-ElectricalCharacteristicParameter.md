## 1.1 底座接口说明
| 编号 | 接口 | 定义 | 功能 | 备注 |
|:----:|:-----------:|:---------:|:-----------------:|:----------------:|
| 1 | 开关 | 电源开关 | 控制电源开关 | 带指示灯（通电时亮） |
| 2 |STOP | STOP | 紧急停止电路接口 | |
| 3 | USB3.2 | USB3.2*2 | 外部设备或U盘连接 | |
| 5 | HDMI | |连接显示器 | |
| 6 | 串口 | |用于收发串口数据的通信接口 | |
| 7 | I2C接口 | |连接I2C设备 | |
| 8 | USB2.0 | USB2.0 | 外部设备或U盘连接 | |
| 9 | Type C | 通信接口 | 与PC通信 | 开发用 |
| 10 | 电源输入接口 | DC24V输入 | DC24V输入 | |

#### 1 电源开关：控制主电源输入的通断，关闭时控制器也断电。

#### 2 紧急停止端子：连接紧急停止按钮盒，用于控制机器人紧急停止。
> **注意**：机器人运行过程中必须连接急停开关，并确保急停开关电路处于连接状态。

#### 3 USB3.2接口：使用USB 3.0串行总线标准进行数据连接用的接口，用户可以使用USB接口拷贝程序文件或者连接鼠标键盘等外设。

#### 5 HDMI显示接口：用户可以通过连接Micro HDMI显示接口，在其他设备上显示操作界面。

#### 6 UART串口：用于发送和接收串行数据的通信接口。

#### 7 I2C接口：用于连接I2C设备的接口。

#### 8 USB-C接口：使用USB 2.0串行总线标准进行数据连接用的接口，用户可以使用USB接口拷贝程序文件或者连接鼠标键盘等外设。此带OTG功能的USB2.0接口由MP8759和AW32901控制器供电。

#### 9 Type C ：用于与PC通信，供开发使用。

#### 10 电源输入接口：此接口接DC24V电源适配器接口。
